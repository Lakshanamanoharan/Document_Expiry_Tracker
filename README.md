A smart web application that helps you track and manage document expiration dates with automatic OCR-based date detection and timely notifications.

Features
1. Multi-User Authentication - Secure login system with user isolation and document privacy
2. Automatic Date Detection - Upload documents (images/PDFs) and let OCR extract expiry dates automatically
3. Manual Entry - Add documents with custom expiry dates, issue dates, and tags
4. Interactive Dashboard - View all documents with filtering, sorting, and search capabilities
5. Smart Notifications - Get alerts for documents expiring soon (7, 14, 30 days)
6. Document Preview - In-app viewer for uploaded documents
7. Custom Tagging - Organize documents with custom tags and categories
8. Analytics Dashboard - Visual insights into document status and expiry trends
9. Bulk Actions - Delete or update multiple documents at once
10.Export Reports - Download document data as CSV for external use
11.Modern UI - Beautiful glassmorphism design with dark mode aesthetics

Tech Stack
Backend: Flask (Python)
Database: SQLite
OCR: Tesseract OCR + pytesseract
Image Processing: Pillow (PIL)
PDF Handling: PyMuPDF
Date Parsing: dateutil
Frontend: HTML5, CSS3, JavaScript
Authentication: Session-based with password hashing
